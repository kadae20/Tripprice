# Hook: SessionStart

## 목적
세션 시작 시 편집국 운영 원칙을 리마인드하고, 현재 운영 상태를 로드한다.

## 실행 파일
.claude/hooks/scripts/session-start.sh

## 실행 내용 (가볍게)

1. **운영 리마인더 출력**
   - 편집국 절대 규칙 5가지 요약
   - 현재 워크플로우 단계 안내

2. **상태 파일 로드 (있는 경우)**
   - state/campaigns/ — 진행 중인 캠페인 목록
   - state/coverage/ — 최근 업데이트된 커버리지 요약

3. **출력 형식**
   ```
   [Tripprice 편집국] 세션 시작
   ── 절대 규칙: 스팸/도어웨이/중복 복제 금지 ──
   진행 중 캠페인: [있으면 목록]
   커버리지 경고 호텔: [있으면 목록]
   ```

## 주의사항
- 무거운 작업 금지 (파일 대량 읽기, 네트워크 요청)
- 출력은 5줄 이내로 간결하게
- 상태 파일이 없으면 조용히 건너뜀
