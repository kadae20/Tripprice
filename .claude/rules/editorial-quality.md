# Rule: 편집 품질 기준

## 발행 전 필수 통과 게이트 (모두 충족해야 Publish 가능)

- [ ] 하나의 명확한 검색 의도(search intent)가 있다
- [ ] 첫 문단에서 "누구를 위한 글인지"가 분명하다
- [ ] 호텔 나열이 아니라 "선택 기준"이 먼저 나온다
- [ ] 각 호텔마다 "왜 추천하는지" 논리가 있다
- [ ] 장점뿐 아니라 아쉬운 점/주의점이 포함된다
- [ ] 키워드는 자연스럽게 사용되고 반복 남발되지 않는다
- [ ] 기존 글과 차별점(관점/동선/예산/페르소나)이 분명하다
- [ ] thin affiliate처럼 보이지 않는다 (실질적 부가가치 있음)

## 필수 구조 요소

모든 호텔 의사결정형 글은 아래 섹션을 포함해야 합니다:

1. **빠른 결론 요약 박스** — 상단에 배치, 핵심 선택 기준과 결론 요약
2. **이 글이 필요한 사람** — 타겟 독자 명시 (1~2줄)
3. **선택 기준 섹션** — 비교 전에 "무엇을 기준으로 골랐는가" 설명
4. **호텔별 분석** — 추천 대상 / 장점 / 단점 / 위치·동선 / 주의점
5. **FAQ** — 3개 이상, 실제 검색 의도 기반
6. **CTA 블록** — 자연스럽게 삽입, 정보 흐름 방해 금지
7. **내부 링크** — 허브 글/도시 가이드/관련 주제 최소 2개

## 금지 패턴

- "X호텔은 훌륭합니다. Y호텔도 좋습니다." → 나열형 금지
- 공급자 설명 복사+붙여넣기 수준의 재조합 → 금지
- 모든 호텔이 장점만 있고 단점 없는 구성 → 금지
- 근거 없는 "최고", "가성비 최강" 단정 표현 → 금지

## 데이터 고지 문구 (모든 글 필수 포함)

> 가격·혜택·환불 규정은 시기에 따라 변동될 수 있으며,
> 최종 조건은 예약 페이지에서 직접 확인하시기 바랍니다.
