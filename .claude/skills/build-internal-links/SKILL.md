# Skill: build-internal-links

## 역할
글에 삽입할 내부 링크를 찾고, 앵커 텍스트를 최적화하여 삽입한다.

## 호출 방식
수동 호출 — insert-affiliate-cta 전 또는 후에 실행.

## 허용 도구
- 파일 읽기/쓰기
- state/ 접근 (발행 글 목록 참조)

## 입력
- state/campaigns/[slug]-humanized.md
- 발행된 글 목록 (state/campaigns/*-published.json)

## 내부 링크 전략

### 링크 유형
1. **허브 → 스포크**: 도시 가이드 → 개별 호텔 글
2. **스포크 → 허브**: 개별 글 → 도시 가이드
3. **스포크 → 스포크**: 관련 테마 글 간 연결

### 앵커 텍스트 규칙
- 목적지 글의 핵심 키워드 반영
- "여기", "클릭", "이 글" 등 무의미한 앵커 금지
- 동일 앵커 텍스트로 다른 URL 연결 금지

### 삽입 위치
- 본문 흐름과 자연스럽게 연결되는 위치
- 글당 최소 2개, 최대 5개
- 첫 문단에 과도한 집중 금지

## 출력
- state/campaigns/[slug]-humanized.md에 인라인 링크 삽입 (수정)
- 삽입된 링크 목록 로그 (state/campaigns/[slug]-links.md)

## 품질 게이트
- 내부 링크 0개면 발행 진행 불가
- 동일 URL에 3개 이상 링크하면 과도 삽입 경고
