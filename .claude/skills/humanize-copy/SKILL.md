# Skill: humanize-copy

## 역할
AI 냄새가 나는 문장을 자연스러운 편집자 문체로 다듬는다.

## 호출 방식
수동 호출 — seo-review 통과 후, insert-affiliate-cta 전에 실행.

## 허용 도구
- 파일 읽기/쓰기

## 입력
- state/campaigns/[slug]-draft.md

## 자연화 작업 기준

### 제거 대상 패턴
- "~할 수 있습니다" 남발 → 간결한 능동형으로 교체
- "최고의", "완벽한", "훌륭한" 근거 없는 수식어 → 제거 또는 근거 추가
- 동일한 문장 구조 반복 (모든 문장이 ~이다. ~이다.) → 변주
- 목록 과다 사용 → 일부 문장형으로 전환
- 어색한 번역체 표현 → 자연스러운 한국어로 교체

### 개선 방향
- 편집자가 직접 취재한 것처럼 구체적인 서술
- 독자에게 직접 말하는 2인칭 활용 적절히
- 첫 문단은 독자의 상황/고민에서 시작
- 주의점/단점은 솔직하되 유용하게 서술

### 톤 기준 (언어별)
- ko: 친근하지만 정보 밀도 높게, 반말 금지
- en: Conversational but informative, avoid passive voice
- ja: 丁寧語 기준, 간결하고 실용적으로

## 출력
state/campaigns/[slug]-humanized.md

## 품질 게이트
- 자연화 전후 문장 수가 20% 이상 줄었으면 → 내용 손실 여부 확인
- "최고", "완벽" 등 수식어가 여전히 근거 없이 남아있으면 → 재처리
