# Skill: seo-review

## 역할
초안의 SEO 적합성을 검토하고 개선 사항을 리포트한다.

## 호출 방식
수동 호출 — write-hotel-article 완료 후, humanize-copy 전에 실행.

## 허용 도구
- 파일 읽기/쓰기
- 웹 검색 (경쟁 글 비교, SERP 확인)

## 입력
- state/campaigns/[slug]-draft.md

## 검토 항목

### 기술적 SEO
- [ ] SEO title 60자 이하, 핵심 키워드 포함
- [ ] meta description 120~155자, CTA 포함
- [ ] slug 소문자-하이픈, 3~5단어
- [ ] H1 1개만 존재, 키워드 포함
- [ ] H2/H3 계층 구조 올바름
- [ ] 대표 이미지 alt 텍스트 작성됨
- [ ] 내부 링크 최소 2개 포함
- [ ] canonical URL 명시

### 콘텐츠 SEO
- [ ] 검색 의도와 글 내용 일치
- [ ] 핵심 키워드 첫 문단에 자연스럽게 포함
- [ ] 키워드 반복 남발 없음 (밀도 2~3% 이내)
- [ ] LSI 키워드(관련 키워드) 자연스럽게 포함
- [ ] FAQ가 실제 검색 질문 기반
- [ ] 글 길이 최소 1,200자 이상 (한국어 기준)

## 출력 (state/campaigns/[slug]-seo-report.md)

```
## SEO 검토 결과
통과: X / 총 Y 항목

## 수정 필요 항목
- [항목]: [문제] → [권장 수정안]

## 발행 적합 여부
[통과 / 조건부 통과(수정 후) / 반려]
```

## 품질 게이트
- H1 2개 이상이면 즉시 반환
- meta description 없으면 반환
- 내부 링크 0개면 반환
