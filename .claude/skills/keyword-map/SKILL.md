# Skill: keyword-map

## 역할
도시/테마/페르소나 단위로 키워드 클러스터를 분석하고, 글 간 키워드 충돌을 방지한다.

## 호출 방식
수동 호출 — 신규 도시/테마 진입 시, 또는 편집장 요청 시.

## 허용 도구
- 웹 검색 (키워드 볼륨/경쟁도 파악)
- 파일 읽기/쓰기
- state/ 폴더 접근

## 입력
- 도시명 또는 테마 (예: "도쿄 호텔", "부산 커플 여행")
- 타겟 언어

## 출력 (state/campaigns/[city-theme]-keyword-map.md)

```
## 핵심 키워드 (허브 글 대상)
[키워드 + 예상 의도 + 볼륨 수준]

## 롱테일 클러스터 (개별 글 대상)
[키워드 그룹별 정리]

## 이미 다룬 키워드 (내부 중복 방지)
[기존 발행글과 연결]

## 발행 우선순위 추천
[1순위~3순위 키워드 + 이유]

## 주의: 아직 다루면 안 되는 키워드
[경쟁 과다 / 데이터 부족 / 품질 미달 이유]
```

## 품질 게이트
- 동일 키워드를 두 글이 나눠 가지면 → 허브-스포크 구조 재설계 필요
- 키워드 맵은 6개월마다 재검토 권장
