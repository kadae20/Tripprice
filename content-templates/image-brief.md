# 템플릿: 대표 이미지 브리프

> 사진이 부족하거나 자체 제작이 필요한 경우 이 템플릿으로 이미지 방향을 정합니다.
> image-editor 에이전트 또는 디자이너에게 전달합니다.

---

## 기본 정보

```
글 제목: [글 제목]
slug: [slug]
이미지 용도: [대표 이미지 / 본문 섹션 이미지 / 비교 카드]
최종 크기: [1200×630px / 1080px 너비 / 기타]
파일 형식: WebP
저장 위치: assets/processed/[hotel-id]/
```

---

## 이미지 유형별 브리프

### A. 호텔 대표 이미지 (사진 있는 경우)
```
호텔명: [이름]
강조 요소: [수영장 / 루프탑 / 로비 / 전경 등]
분위기: [모던 / 클래식 / 아늑함 / 럭셔리]
alt 텍스트: "[호텔명] [특징] [위치]"
```

### B. 위치/동선 인포그래픽 (사진 부족 시)
```
표현할 내용: [역 ↔ 호텔 동선 / 주변 랜드마크 / 지도 기반]
포함할 텍스트: [역명, 도보 시간, 주요 랜드마크]
스타일: 심플한 아이콘 지도, Tripprice 브랜드 색상
alt 텍스트: "[호텔명] 위치 [역명] 도보 [시간]분"
```

### C. 요약 정보 카드 (커버리지 낮은 호텔)
```
포함할 정보: [호텔명 / 가격대 / 위치 / 추천 대상 / 아이콘]
레이아웃: 카드형, 2~3개 병렬 배치 가능
alt 텍스트: "[호텔명] 요약 정보 카드 [도시명]"
```

### D. 비교표 헤더 이미지
```
표현할 내용: 이 글에서 비교하는 호텔 수, 도시명, 주제
스타일: 텍스트 오버레이 + 배경 이미지
alt 텍스트: "[도시명] 호텔 [N]개 비교 [테마]"
```

---

## 워터마크 적용 여부

- [ ] 파트너 정책 확인 완료
- [ ] 워터마크 적용 (우하단, 크기 10% 이하)
- [ ] 워터마크 미적용 (정책 불명확 또는 자체 제작)

---

## 주의사항

- 이미지 가공 전 반드시 `.claude/rules/image-policy-and-processing.md` 확인
- 라이선스 불명확 이미지는 사용 금지
- 완성 이미지는 `assets/processed/[hotel-id]/`에 저장
